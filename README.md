# 🛡️ Cloud Forensics Automation for Retail Breaches

This project detects file tampering in a retail system by comparing current files with a baseline and identifying suspicious changes.

---

## 🚀 Features

- File integrity monitoring  
- Baseline vs current comparison  
- Suspicious activity detection  
- Security alerts with details  
- CSV evidence report generation  

---

## 🛠️ Technologies Used

- Python  
- Streamlit  
- Pandas  

---

## ▶️ How to Run

```bash
git clone https://github.com/lavatoz/Cloud-Forensics-Automation-for-Retail-Breaches.git
cd Cloud-Forensics-Automation-for-Retail-Breaches
pip install -r requirements.txt
streamlit run app.py
